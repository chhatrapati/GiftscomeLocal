Getting Started

After Purchase the module (link below)

Developer Guide documentation are available in zip file.

The script is already manage rosters (i.e. which users are online etc.), and it enables users to chat with each other.
I assume your current application also provides that functionality.
You must also make sure that $_SESSION['username'] and $_SESSION['id'] is being set when your website session begins.
You will understand the logic better after you try the Demo files that I have provided.

Now in your list of users online, add “javascript:chatWith(‘USERNAME’);” function where USERNAME is the username for that particular user who he/she wants to chat with.

Once that is done, edit chat.php and set your database parameters and try your website.

For better understanding, load 2 different browsers (internet explorer, firefox, safari) and point them to with diffrent userid.

And start chat with each other and watch the chat functionality come alive!

Features

  • 1. Share Picture and Files from your device facebook style
  • 2. Auto ping when new messages get
  • 3. Sound notification Get alerted on new message
  • 4. Facebook style chat boxes display on bottom right
  • 5. Load more messages on scroll up
  • 6. Auto ping when new messages get
  • 7. Keeps chat boxes open and stores state (data) even when pages are browsed/refreshed similar to Facebook
  • 8. Displays "Sent at..." after 3 minutes of inactivity
  • 9. Displays "USERNAME says..." & blinks chat boxes when window is not in focus
  • 10. Minimize and close chat boxes
  • 11. Auto-resize of text input box
  • 12. Auto-scrolling of chat text
  • 13. Auto-back-off polling policy (hits the server less-often when chat activity is low)
  • 14. Extremely simple to integrate into existing site
  • 15. Multi chat at same time
  • 16. Friend List
  • 17. Easy to setup
  • 18. Chatbox with inbox messaging
  • 19. User Friendly
  • 20. Eye-catching design.
